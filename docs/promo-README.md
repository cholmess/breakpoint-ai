# Promo Materials

## Files

- **`promo-chatgpt-instructions.md`** — Copy-paste prompt for ChatGPT to generate a blog post or social thread
- **`promo-blog-post.md`** — Full blog post (Markdown)
- **`promo-blog-post.html`** — Same post, HTML for print-to-PDF

## Create PDF

No CLI tools required. Use your browser:

1. Open `docs/promo-blog-post.html` in Chrome, Safari, or Firefox
2. **File → Print** (or Cmd+P)
3. Choose **Save as PDF** (or **Print to PDF**)
4. Save

The HTML includes `@media print` styles for clean output.
